# Grade Import Files

This directory contains uploaded grade import files (CSV/Excel).

## File Naming Convention

Files are automatically named with timestamp and GUID:
- Format: `yyyyMMddHHmmss_GUID.extension`
- Example: `20251204120345_a1b2c3d4e5f6g7h8.xlsx`

## File Retention

- Files are stored permanently unless manually deleted
- Failed imports will have their files automatically removed
- Files can be downloaded from the Grade Index page

## Security Notes

- Only Admin and Lecturer roles can upload files
- Maximum file size: 5 MB
- Supported formats: CSV (.csv) and Excel (.xlsx)
